## 🎹 Interface Controls
Interact with the terminal using standard mouse bindings and structural dashboard widgets:
 * **Left Mouse Click** — Initialize Vector Core selection tracking at target coordinates.
 * **Mouse Drag Pass** — Trace linear trajectory path (Horizontal, Vertical, and Diagonal pathways supported).
 * **Mouse Release** — Commit extracted string sequence to the lexical validation pipeline.
 * **START RUN** — Boots data generation matrices for the active sector layer.
 * **PAUSE DECRYPT** — Halts execution tickers and displays the secure memory masking screen.
 * **CYCLE SYSTEM THEME** — Instantly swaps visual channels between the 4 premium tactical modes.
 * **REBOOT ENGINE** — Flushes local save profiles and restores variables to sector 1 baseline.
